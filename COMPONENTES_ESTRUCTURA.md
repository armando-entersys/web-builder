# Inventario Completo de Componentes - Web Builder

## Resumen Ejecutivo

El proyecto **web-builder** contiene **160 componentes React** organizados en **7 secciones principales**. 
La estructura está diseñada para máxima reutilización, con énfasis en componentes de animación y efectos visuales.
